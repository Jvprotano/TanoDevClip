# TanoDev Clip

Developer-first clipboard manager with local dev tools.

## Goals

- Fast desktop clipboard manager.
- Local-first storage.
- Developer-oriented classification and actions.
- Built-in dev tools like GUID, CPF, CNPJ, Lorem Ipsum and JWT decoder.
- Open source.

## Tech Stack

- .NET 10
- WPF
- WebView2
- React + Vite + TypeScript
- SQLite
